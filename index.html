Image Base64 Transfer

純前端工具。圖片與 Base64 文字轉換都在瀏覽器本機完成；GitHub 同步功能會呼叫 GitHub REST API。

建議流程：圖片 → 匯出 Base64 txt → Export to GitHub → 另一端 Import from GitHub → 還原圖片。GitHub Token 不會被存進瀏覽器儲存空間，但會存在目前頁面的記憶體中直到重新整理。

格式說明

本工具輸出的 txt 格式如下。Unity 版工具與這個網頁版可以互相讀取。

UNITY_IMAGE_BASE64_TRANSFER_V1 CREATED_UTC=2026-05-19T00:00:00Z SOURCE_MODE=RawImageFileEncode SOURCE=example.png FORMAT=PNG WIDTH=1024 HEIGHT=1024 BYTE_LENGTH=123456 BASE64_LENGTH=164608 LINE_LENGTH=8000 SHA256=... BASE64_BEGIN iVBORw0KGgoAAAANSUhEUgAA... BASE64_END

注意：Base64 不是壓縮，通常會比原始檔案大約三分之一。大圖建議用 txt 檔或 GitHub 同步，不要直接貼到輸入框。
